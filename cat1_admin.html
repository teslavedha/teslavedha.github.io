KTET Category1 — Admin Panel (All Subjects)

This admin stores data in the browser's localStorage. You can manage videos (10), notes PDFs (5) and MCQ PDFs (5) per subject.
Usage notes:
  • For YouTube links use standard watch URL: https://www.youtube.com/watch?v=VIDEO_ID. Admin extracts ID automatically.
  • PDF URLs should be direct links (raw GitHub file or hosted file).
  • After saving, open the subject user pages to see changes:
    • Main: https://teslavedha.github.io/cat1_SUBJECT.html
    • Videos: https://teslavedha.github.io/cat1_SUBJECT_videos.html
    • Notes: https://teslavedha.github.io/cat1_SUBJECT_notes.html
    • MCQ: https://teslavedha.github.io/cat1_SUBJECT_mcq.html
Admin data is stored locally in your browser. To share data across devices, ask to upgrade to Firebase or Google Sheets.
